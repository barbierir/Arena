# Arena Scaffold

Minimal browser game scaffold with a Node.js + Express backend and plain HTML/CSS/JS frontend.

## Folder tree
```text
.
├── package.json
├── server
│   └── index.js
└── public
    ├── app.js
    ├── challenge.html
    ├── game.html
    ├── index.html
    ├── recruit.html
    ├── styles.css
    └── summary.html
```

## How to run locally
```bash
npm install
npm start
```

Optional dev mode:
```bash
npm run dev
```

Then open `http://localhost:3000`.
